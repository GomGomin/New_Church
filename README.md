# New_Church

## ⭐ 개요

- **제작 의도 : 클라이언트의 요구사항에 맞춰 웹 페이지 제작**

## 📱 주요기능
- **교회홍보 , 정보제공, 온라인 예배 제공, 커뮤니티 게시판, Chat GPT**

## 🛠️ 사용 기술 및 라이브러리

- **사용 기술 : JAVA, Spring, MySQL, AJAX, BootStrap, MyBatis**
- **프로그램 : Eclips, HeidiSQL, Git, VsCode**
- **라이브러리 : jQuery, 카카오API, Chat GPT API**
